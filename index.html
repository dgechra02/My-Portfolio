Kanban Board

About the project

I developed a Kanban board application focused on task organization and management. The app provides an intuitive interface for categorizing tasks into different sections. Each section represents a stage in the workflow, making it easy to track progress. The application is ideal for managing team projects effectively. It ensures clarity and structure in task management processes. Future enhancements will include drag-and-drop functionality and real-time updates.

Todo App

About the project

I built an todo app for task listing and management. Through this app one can write their daily tasks and add in the list. Once a task is completed, they can either send it in the completed task tab by clicking on the check mark button or they can directly delete the task. In the completed tab, there are the tasks that are completed. Once you are done with all the task, you can also delete they from this tab. Tasks saves in the local storage, so they remains there even after closing or refreshing the page.
